# 국립공원 입장료 초이스 워크플로우 가이드

## 워크플로우 개요

### Step 1: 통합 옵션에서 템플릿 생성

**위치**: `통합 옵션` > `초이스 관리` 탭

1. **그룹 추가** 클릭
2. 그룹 정보 입력:
   - 그룹 이름 (영문): `national_park_fee`
   - 그룹 이름 (한글): `국립공원 입장료`
   - 초이스 타입: `single` (단일 선택)
   - 필수 여부: `true`
   - 최소/최대 선택 수: `1`

3. **템플릿 추가** 버튼으로 모든 국립공원 옵션 추가:
   - 미국 거주자 ($8)
   - 비 거주자 ($100)
   - 애뉴얼 패스 구매자 ($250)
   - 애뉴얼 패스 동행자 ($0)
   - **그랜드캐년 입장료** (예시)
   - **자이언캐년 입장료** (예시)
   - **브라이스 캐년 입장료** (예시)
   - **요세미티 입장료** (예시)
   - **세쿼이아 입장료** (예시)
   - ... (모든 가능한 국립공원)

**결과**: 통합 옵션에 "국립공원 입장료" 템플릿 그룹이 생성되고, 모든 국립공원 옵션이 포함됩니다.

---

### Step 2: 상품 편집에서 템플릿 불러오기

**위치**: `상품 편집` > `초이스 관리` 탭

1. **템플릿에서 불러오기** 버튼 클릭
2. "국립공원 입장료" 템플릿 선택
3. 템플릿이 불러와지면 초이스 그룹이 생성됩니다.

**결과**: 상품에 "국립공원 입장료" 초이스 그룹이 추가되고, 템플릿의 모든 옵션이 포함됩니다.

---

### Step 3: 상품별로 필요한 옵션만 선택/수정

**위치**: `상품 편집` > `초이스 관리` 탭

#### 옵션 삭제
- 각 옵션 옆의 **삭제 버튼** (🗑️) 클릭
- 불필요한 공원 옵션 제거

#### 옵션 추가
- **옵션 추가** 버튼 클릭
- 필요한 경우 새로운 옵션 추가

#### 옵션 수정
- 옵션 이름, 가격, 설명 등 수정 가능

#### 예시: 밤도깨비 투어
```
국립공원 입장료 그룹:
├── 미국 거주자 ($8)
├── 비 거주자 ($100)
├── 애뉴얼 패스 구매자 ($250)
├── 애뉴얼 패스 동행자 ($0)
└── 그랜드캐년 입장료 (필요한 경우만 추가)
    (자이언, 브라이스, 요세미티 등은 삭제)
```

#### 예시: 그랜드서클 투어
```
국립공원 입장료 그룹 1: 그랜드캐년 입장료
├── 미국 거주자 ($8)
├── 비 거주자 ($100)
├── 애뉴얼 패스 구매자 ($250)
└── 애뉴얼 패스 동행자 ($0)

국립공원 입장료 그룹 2: 자이언캐년 입장료
├── 미국 거주자 ($8)
├── 비 거주자 ($100)
├── 애뉴얼 패스 구매자 ($250)
└── 애뉴얼 패스 동행자 ($0)

국립공원 입장료 그룹 3: 브라이스 캐년 입장료
├── 미국 거주자 ($8)
├── 비 거주자 ($100)
├── 애뉴얼 패스 구매자 ($250)
└── 애뉴얼 패스 동행자 ($0)
```

**방법**: 템플릿을 3번 불러온 후, 각 그룹의 이름을 수정하고 불필요한 옵션 삭제

## 워크플로우 요약

```
1. 통합 옵션 - 초이스 관리
   └── "국립공원 입장료" 템플릿 그룹 생성
       └── 모든 국립공원 옵션 등록 (그랜드캐년, 자이언, 브라이스, 요세미티, 세쿼이아 등)

2. 상품 편집 - 초이스 관리
   └── 템플릿에서 "국립공원 입장료" 불러오기
       └── 모든 옵션이 포함된 초이스 그룹 생성

3. 상품별 커스터마이징
   ├── 불필요한 공원 옵션 삭제
   ├── 필요한 공원 옵션만 남기기
   ├── 옵션 이름/가격 수정
   └── 동적 가격 설정 (상품별, 날짜별)
```

---

## 장점

✅ **템플릿 재사용**: 한 번 만든 템플릿을 여러 상품에서 사용
✅ **상품별 커스터마이징**: 각 상품에서 필요한 옵션만 선택
✅ **유지보수 용이**: 템플릿 수정 시 상품별 초이스는 독립적으로 관리
✅ **일관성 유지**: 공통 옵션(미국 거주자, 비 거주자, 애뉴얼 패스)은 템플릿에서 관리

---

## 주의사항

⚠️ **템플릿과 상품 초이스는 독립적**
- 템플릿을 불러온 후에는 상품별 초이스로 독립적으로 관리됩니다.
- 템플릿을 수정해도 이미 불러온 상품 초이스에는 영향이 없습니다.
- 상품 초이스를 수정해도 템플릿에는 영향이 없습니다.

⚠️ **동적 가격은 상품별로 설정**
- 템플릿의 기본 가격은 참고용입니다.
- 실제 가격은 동적 가격 관리에서 상품별, 날짜별로 설정해야 합니다.

## FAQ

**Q: 템플릿에서 옵션을 삭제하면 이미 불러온 상품 초이스에도 영향이 있나요?**
A: 아니요. 템플릿을 불러온 후에는 상품별 초이스로 독립적으로 관리되므로 영향이 없습니다.

**Q: 상품에서 옵션을 삭제하면 템플릿에도 영향이 있나요?**
A: 아니요. 상품 초이스와 템플릿은 완전히 독립적입니다.

**Q: 여러 상품에서 같은 템플릿을 사용할 수 있나요?**
A: 네, 가능합니다. 각 상품에서 템플릿을 불러온 후 개별적으로 수정할 수 있습니다.

**Q: 템플릿에 새로운 공원을 추가하면 어떻게 되나요?**
A: 템플릿에만 추가되고, 이미 불러온 상품 초이스에는 자동으로 추가되지 않습니다. 필요하면 각 상품에서 수동으로 추가하거나 템플릿을 다시 불러와야 합니다.
