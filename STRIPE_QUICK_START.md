# 🚀 Stripe 빠른 시작 가이드 (5분 안에!)

## ⚡ 빠른 설정 3단계

### 1단계: Stripe 계정에서 키 가져오기 (2분)

1. https://dashboard.stripe.com 접속
2. 로그인 후 왼쪽 메뉴에서 **"Developers"** 클릭
3. **"API keys"** 클릭
4. **"Test mode"** 토글이 켜져 있는지 확인 (초록색)
5. **"Publishable key"** 복사 → `pk_test_...`로 시작
6. **"Secret key"**의 "Reveal test key" 클릭 → `sk_test_...`로 시작

### 2단계: 환경 변수 파일 만들기 (1분)

1. 프로젝트 폴더에 `.env.local` 파일 생성
2. 다음 내용 복사해서 붙여넣기:

```env
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=여기에_pk_test_붙여넣기
STRIPE_SECRET_KEY=여기에_sk_test_붙여넣기
```

3. 파일 저장!

### 3단계: 서버 재시작하고 테스트! (2분)

1. 터미널에서 서버 재시작:
   ```bash
   # Ctrl + C로 서버 중지 후
   npm run dev
   ```

2. 브라우저에서 예약 플로우 진행
3. 결제 단계에서 테스트 카드 입력:
   - **카드 번호**: `4242 4242 4242 4242`
   - **만료일**: `12/25`
   - **CVV**: `123`
   - **이름**: 아무 이름
4. "예약 완료" 클릭
5. 성공! 🎉

---

## ❓ 문제가 생겼다면?

**"Invalid API Key" 에러?**
- `.env.local` 파일 위치 확인 (프로젝트 루트에 있어야 함)
- 서버 재시작했는지 확인
- 키 앞뒤 공백 없이 정확히 복사했는지 확인
